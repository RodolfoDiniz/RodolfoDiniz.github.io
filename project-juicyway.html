Juicy Way

Juicy Way is a galactic game that is set on a fruity galaxy. It is a free endless runner 2D experience created by a designer and a developer.

  • 2020

The Flight Plan

At Apple Developer Academy we were challenged to develop and publish a hyper casual game and to apply the freemium monetization so we would be able to track the results from it. Some general parameters that we had to follow: hyper-casual game; preferably 2D with few visual elements and interactions; rapid phases/interactions lasting from ten seconds to two minutes.

Lifting Off

Before creating this endless space fruity chaser, we started by gathering some references of casual games and graphic elements. By using forced connections, we were able to bring two unconventional topics together, such as Fruit and Space.

The interface of the menu revolves around the concept of having the game just one play-tap away. To achieve that, it was designed to work as a layer that goes on top while the intergalactic spaceship is always traveling through the Juicy Way right behind it.

Safe Landing

In the first week, we developed the basic mechanics and submitted to Test Flight; in the second, we included monetization and analytics, and submitted for publication; on the third and final week the focus was on user acquisition, campaign development and metrics tracking.

Besides the assets and visual identity, I projected the menu and the HUD, and coded both interfaces using SwiftUI while André Tosin developed the mechanics using SpriteKit.
